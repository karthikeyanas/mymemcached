MYMEMCACHED
==========

This assumes that you have libmemcached installed. If not install
libmemcached before building Mymemcached.

wget https://launchpad.net/libmemcached/1.0/1.0.18/+download/libmemcached-1.0.18.tar.gz

1. Run "make"
2. Run "./startmymemcached" to start the server.
3. Run "./startTests" to run tests that runs some unit test on
   mymemcached server.
4. Run "./stopmymemached" to stop the server.
